# Wanderlust – Full Stack Hotel Booking Platform
## 📌 Project Overview
Wanderlust is a full stack hotel booking web application that allows users to explore hotel listings, register/login securely, and manage bookings.

The application follows proper MVC architecture and includes authentication, REST APIs, and database integration.

---

## 🚀 Tech Stack

Frontend:
- HTML
- CSS
- JavaScript
- EJS (if used)

Backend:
- Node.js
- Express.js

Database:
- MongoDB
- Mongoose

Authentication:
- Session / Passport.js (if used)

---

## ✨ Features

- User Registration & Login
- Create / Edit / Delete Hotel Listings
- Booking Management
- Secure Authentication
- MVC Architecture
- Middleware-based route protection

## ⚙️ How to Run Locally

1. Clone the repository
2. Run `npm install`
3. Create `.env` file for environment variables
4. Run `node app.js`
5. Open `localhost:3000`
